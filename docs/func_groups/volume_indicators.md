# Volume Indicator Functions
### AD - Chaikin A/D Line
```python
real = AD(high, low, close, volume)
```

### ADOSC - Chaikin A/D Oscillator
```python
real = ADOSC(high, low, close, volume, fastperiod=3, slowperiod=10)
```

### OBV - On Balance Volume
```python
real = OBV(close, volume)
```
